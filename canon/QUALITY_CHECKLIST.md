# Quality Checklist (Before Sharing With Counsel/Court)

- [ ] Every major claim has an exhibit anchor (EXH-###)
- [ ] Child identifiers are redacted
- [ ] Tone is neutral (no insults/motive claims)
- [ ] Chronology is date-ordered
- [ ] Contradictions are noted as “unknown” vs argued
- [ ] Requests are specific and child-centered
- [ ] Raw files are stored privately, not in GitHub
