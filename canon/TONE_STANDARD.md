# Tone Standard (Court-Ready Neutral)

## Write like this
- “On [date], [event] occurred.”
- “The message states: ‘[short quote]’”
- “I requested [specific action] at [time].”
- “The exchange did not occur as scheduled.”

## Avoid writing like this
- “They did this to hurt me.”
- “They’re lying / crazy / malicious.”
- “Everyone knows they…”
- “They always / never…” (unless you have data to prove it)

## The Three-Layer Sentence
When you must describe impact:
1) Fact: “The exchange did not occur on [date].”
2) Child impact: “This affected [CHILD]’s routine and created uncertainty.”
3) Remedy: “I request a clear make-up time process within 7 days.”

## Prohibited content
- insults
- profanity
- threats
- diagnoses
- speculation about motives
