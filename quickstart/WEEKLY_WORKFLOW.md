# Weekly Workflow (30 minutes)

1) Add new events (Mirror)
- Parenting-time events
- Communications (logistics only)
- Any deviations from schedule

2) Update exhibits
- Add exhibit IDs for new records
- Update `EXHIBIT_LOG.csv`

3) Update chronology
- Keep it date-ordered
- Link every entry to exhibits

4) Write weekly summary (Water)
- What repeated?
- What improved?
- What is still unclear?

5) If needed, update remedies (Fire)
- Keep remedies concrete
- Avoid punishment framing
